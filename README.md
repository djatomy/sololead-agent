# SoloLead Agent

**オリジナル版画アーティストのためのニッチ需要発掘＆Instagram DM自動化ツール**

自分の Instagram を作品ポートフォリオのメイン展示場として活用しながら、本当に版画を欲しがるニッチな層（新築・結婚祝い・記念日・コレクションなど）を自動で探し、自然な DM で商談をスタートさせるツールです。

## 📌 主な機能（MVP）

- 🎨 **作品プロフィール登録** - Instagram URL とスタイルを一度登録
- 🔍 **ニッチな需要の自動発掘** - Instagram・X・Google で顧客シグナルを検索
- 💬 **パーソナライズされた DM 生成** - AI が自然な初回 DM を自動作成
- 📊 **リード管理ダッシュボード** - マッチ度スコア付きリストとステータス追跡

## 🚀 クイックスタート

### 前提条件
- ブラウザ（Chrome / Firefox / Safari 最新版）
- Instagram アカウント
- AI API キー（Gemini / Claude / OpenAI いずれか）

### セットアップ手順

1. **リポジトリをクローン**
   ```bash
   git clone https://github.com/djatomy/sololead-agent.git
   cd sololead-agent
   ```

2. **ブラウザで開く（方法1: 直接開く）**
   ```bash
   open index.html  # macOS
   # または
   start index.html  # Windows
   # または
   ファイルマネージャーで index.html をダブルクリック
   ```

3. **ブラウザで開く（方法2: ローカルサーバー経由 - 推奨）**
   ```bash
   # Python 3 がインストール済みの場合
   python -m http.server 8000
   
   # その後、ブラウザで http://localhost:8000 を開く
   ```

   または npm で実行:
   ```bash
   npm install
   npm start
   ```

4. **プロフィール情報を入力**
   - Instagram URL を入力
   - 作品スタイル（最大200文字）を説明
   - 「情報を保存」ボタンをクリック

5. **「AIでニッチな需要を探す」をクリック**
   - マッチ度スコア付きリードが表示されます
   - DM 文をコピーして、Instagram で手動送信
   - 「このリードを保存」でリード管理

## 🧪 テスト方法

### テスト用ダッシュボードを開く
```
http://localhost:8000/test.html
```

または直接開く: `test.html` をブラウザにドラッグ&ドロップ

### テストの実行
1. **「すべてのテストを実行」** - 全テストスイートを実行
2. **「ユニットテスト」** - ユーティリティ関数のテスト
3. **「統合テスト」** - ストレージやビジネスロジックのテスト

### テスト項目
- ✅ **ユーティリティ関数** - URL バリデーション、HTML エスケープ、UUID 生成
- ✅ **ストレージ管理** - localStorage の読み書き、プロフィール・リード管理
- ✅ **ビジネスロジック** - DM 生成、マッチスコア計算
- ✅ **UI インタラクション** - メッセージ表示、入力値キャプチャ、文字数カウント
- ✅ **エラーハンドリング** - 不正入力処理、JSON パースエラー処理

### 開発者モード
メインアプリ（index.html）で「🔧 開発者モード」ボタンをクリック:
- **バリデーションテスト** - 入力値検証のテスト
- **ストレージテスト** - localStorage 動作確認
- **リード生成テスト** - DM 生成ロジックのテスト
- **全データ削除** - 保存データをクリア
- **デバッグ情報エクスポート** - 現在の状態を JSON で出力

## 📋 使用フロー

```
① プロフィール登録
  ↓
② 「Find New Leads」ボタンで検索
```

## ⚠️ 重要な注意事項

- 🔒 **Instagram 利用規約の厳守** - スクレイピングは禁止されています。公開情報のみを使用してください
- 👤 **DM送信は必ず手動** - このツールは AI が提案文を生成するだけです。最終送信は常に手動で行ってください 
- 📧 **スパム扱い回避** - 一度に大量の DM を送信しないようご注意ください（毎日3〜5件程度を推奨）
- 🔐 **個人情報保護** - 取得した顧客情報は責任持って管理してください
